Projet 6

Domaine : Programmation Web Backend & Frontend Contexte : Création d'une application de messagerie instantanée avec gestion de fichiers Technologies : PHP (Sessions, manipulation de fichiers), HTML5, CSS3 (Flexbox) Année : 2025

Description

Ce projet a été réalisé dans le cadre du BUT MMI lors d'un module d'initiation au développement web dynamique avec PHP. L'objectif était de concevoir une application de messagerie instantanée fonctionnelle, permettant de comprendre les interactions entre un formulaire client, un script de traitement serveur et un système de stockage de données.
